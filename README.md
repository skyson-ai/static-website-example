# Static Website Example

A clean static website example — ideal for learning HTML, CSS, and JavaScript fundamentals.

## Overview

A minimal, well-structured static website demonstrating modern web development best practices. Originally created for Cloud Academy labs.

## Features

- Clean semantic HTML5
- Responsive CSS layout
- Vanilla JavaScript interactions
- Zero build step required
- Fast loading — no framework overhead

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Responsive styling
- **JavaScript** — Client-side interactivity

## Getting Started

```bash
git clone https://github.com/skyson-ai/static-website-example.git
cd static-website-example
```

Open `index.html` in your browser.
